# Sun Powder Brand Assets

This folder contains brand guidelines and assets for Sun Powder. Duplicate this folder into any new project to ensure brand consistency from the start.

**Website reference:** [drinksunpowder.com](https://drinksunpowder.com)

---

## Folder Structure

```
brand-assets/
├── README.md          ← You are here
├── colors/            ← Color palette & CSS variables
├── fonts/             ← Söhne & Söhne Breit font files
├── backgrounds/       ← Solid colors with tints/hues
├── product-shots/     ← Product photography
├── logos/             ← Logo files
└── pricing/           ← Pricing matrix (one-time & subscription)
```

---

## Quick Reference

### Colors
Primary: `#FF4000` (Orange) · `#FF9C00` (Amber) · `#041C72` (Navy)

See `/colors/README.md` for full palette.

### Typography
- **Söhne Breit** — Headlines
- **Söhne** — Descriptors, subheads, body, buttons

See `/fonts/README.md` for usage details.
